# Khaatabook Web App

## Overview
Khaatabook Web App helps users manage financial transactions easily.

## Features
- Secure user authentication
- Add, edit, and delete transactions
- Responsive UI
- Secure data storage

## Tech Stack
- HTML, CSS, JavaScript, Tailwind CSS
- Node.js, Express.js
- MongoDB
- JWT Authentication

## Usage
- Register/Login
- Manage transactions
- Track balances
- Logout securely
